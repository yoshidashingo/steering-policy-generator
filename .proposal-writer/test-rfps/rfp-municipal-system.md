- **調達番号**: さくら市公告第2026-DX-003号

- **質問受付期限**: 2026-02-20
- **質問回答公開**: 2026-02-28

- **契約期間**: 2026年6月〜2028年3月（開発12ヶ月＋運用保守10ヶ月）

## 2. 事業の背景

さくら市（人口12万人）では、「さくら市デジタル変革推進計画（令和6年度〜令和10年度）」に基づき、住民サービスのデジタル化を推進している。現在、各種行政手続き（転入届、各種証明書発行、施設予約等）は窓口対応が中心であり、以下の課題がある：

1. 窓口混雑による住民の待ち時間増大
2. 紙ベースの申請による処理時間の長期化
3. 開庁時間内のみの対応による利便性の制限
4. 手続きごとに異なるシステムによる情報の分散

## 3. 業務内容

### 3.1 システム概要
住民がオンラインで各種行政手続きを完結できるポータルシステムを構築する。

### 3.2 必須要件

#### A. 住民向け機能
- REQ-A01: マイナンバーカードによる本人認証（JPKI準拠）を実装すること【必須】
- REQ-A02: 転入・転出届のオンライン申請機能を提供すること【必須】
- REQ-A03: 住民票・印鑑証明等の各種証明書のオンライン申請機能を提供すること【必須】
- REQ-A04: 公共施設（図書館、体育館、会議室）のオンライン予約機能を提供すること【必須】
- REQ-A05: 申請状況のリアルタイム追跡機能を提供すること【必須】
- REQ-A06: 多言語対応（日本語、英語、中国語、韓国語）を実装すること【必須】
- REQ-A07: 住民への通知機能（メール、プッシュ通知）を提供すること【必須】

#### B. 職員向け機能
- REQ-B01: 申請の受付・審査・承認ワークフローを実装すること【必須】
- REQ-B02: 申請データの統計・分析ダッシュボードを提供すること【必須】
- REQ-B03: 既存住民基本台帳システムとのデータ連携を実装すること【必須】
- REQ-B04: 監査ログ（操作履歴）の記録・閲覧機能を実装すること【必須】

#### C. 基盤要件
- REQ-C01: ガバメントクラウド（AWS または Azure）上に構築すること【必須】
- REQ-C02: ISMAP認証を取得したサービスを利用すること【必須】
- REQ-C03: 個人情報保護法および自治体情報セキュリティポリシーに準拠すること【必須】
- REQ-C04: 年間稼働率99.9%以上を保証すること【必須】
- REQ-C05: 同時接続3,000ユーザー以上に対応すること【必須】
- REQ-C06: JIS X 8341-3:2016 レベルAA準拠のアクセシビリティを確保すること【必須】
- REQ-C07: 災害時のBCP対応（マルチリージョン構成）を実装すること【必須】

### 3.3 任意要件
- REQ-D01: AIチャットボットによる手続き案内【加点】
- REQ-D02: LINEミニアプリとの連携【加点】
- REQ-D03: やさしい日本語対応【加点】
